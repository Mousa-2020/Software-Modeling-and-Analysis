## 📖 About The Project (عن المشروع)

**English:**
An interactive, mobile-first web application developed for the **Software Modeling and Analysis (SMA)** course laboratory. This project transforms the traditional Python lab manual into an engaging platform where students can read theories, run Python code directly in the browser, and test their knowledge through interactive challenges and a comprehensive final exam. 

**العربية:**
تطبيق ويب تفاعلي (مصمم خصيصاً للهواتف الذكية) تم تطويره لمختبر مادة **"نمذجة وتحليل البرمجيات"**. يحول هذا المشروع ملزمة مختبر بايثون التقليدية إلى منصة تفاعلية تتيح للطلاب قراءة الشرح، كتابة وتجربة أكواد بايثون مباشرة داخل المتصفح، واختبار معلوماتهم عبر تحديات برمجية وامتحان شامل.

---

## ✨ Features (المميزات)

* 📱 **Mobile-First Design:** A clean, dark-mode UI optimized for seamless studying on mobile devices.
* 📚 **7 Comprehensive Lessons:** Covering everything from Python Variables to Object-Oriented Programming (OOP) and Exceptions.
* ▶️ **In-Browser Python Simulator:** Write and execute basic Python code directly in the browser without needing a backend server!
* 🏆 **Gamified Learning:** Track your progress with a built-in scoring system and completion rings (saves data locally).
* 📋 **Cheat Sheet:** A quick-reference guide for all essential Python syntax.
* 📝 **Final Exam:** A mix of Multiple Choice Questions (MCQs) and coding challenges to test overall understanding.

---

## 🛠️ Built With (التقنيات المستخدمة)

This project was intentionally built using core, foundational web technologies to solidify front-end basics:
* **HTML5** (Semantic structure)
* **CSS3** (Custom properties, Flexbox, Grid, Animations)
* **Vanilla JavaScript** (DOM manipulation, local storage, custom Python code evaluator)
* *Zero external frameworks or libraries.*
